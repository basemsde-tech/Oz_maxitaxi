☁️ OneDrive Sync

Sign in with Microsoft

Your invoices are saved securely in your own OneDrive account. No third-party servers — your data stays yours.

Your Azure App ID (Client ID)

A free Azure App Registration is needed so this app can access your OneDrive. Takes ~3 minutes to set up. Follow the steps below.

Paste your Azure Application (client) ID above, then tap Sign In.
How to get your free Client ID
1. Go to portal.azure.com (free Microsoft account)
2. Search "App registrations" → New registration
3. Name: OZ Maxi Taxi · Account type: Personal Microsoft accounts only
4. Redirect URI: Single-page application (SPA)
    → URL: https://YOUR-NETLIFY-URL.netlify.app
5. Click Register → copy the Application (client) ID
6. Paste it above → tap Sign In
